# UAE Health Insurance Training Academy
### Created by Rana Musab Bin Tariq

A training resource for healthcare professionals (doctors, pharmacists, physiotherapists)
transitioning into UAE health insurance and TPA roles.

## Run Locally
```bash
pip install streamlit
streamlit run app.py
```

## Deploy on Streamlit Cloud
1. Push this folder to a GitHub repo
2. Go to share.streamlit.io
3. Connect GitHub, select repo, point to app.py
4. Deploy — live in ~2 minutes

## What's Covered (13 screens)
- UAE Insurance Landscape (DHA, HAAD, MOH, CII principles, 2025 updates)
- Exclusions Reference (42 DHA + 56 HAAD + Northern Emirates, searchable)
- Claims & Adjudication (PA workflow, PD-05-2025, adjudication logic, denial codes, procedure criteria)
- Medical Coding & DRG (CPT, ICD-10, HCPCS, IR-DRG grouper, HCC/RAF)
- Pre-existing Conditions & Underwriting (UNPEC, file audit, payer-specific rules)
- Inpatient & Emergency (IP auth, 24-hr rule, DRG payment, daycase rules)
- Internal Audit & FWA (risk-based audit, upcoding schemes, documentation)
- Networks & Reimbursement (how networks work, direct billing, reimbursement process)
- Specialty Rules (physiotherapy, pharmacy, maternity, dental)
- Case Studies (10 real-world scenarios)
- Knowledge Quiz (20 questions, 80% pass mark)
- Resources & Certificate
